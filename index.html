Why learn Linear Programming?

Let's learn about linear programming!!!! I know you're already SO excited about it, but just in case you're one of those people who needs extra convincing, here are some great reasons to learn this powerful technique:

  • Simulating partial equilibrium problems
  • Optimizing shipping/transport networks
  • Finding minimum graph cuts
  • Optimizing statistical models

Any problem that can be modeled as an objective function to be optimized (cost, sum of squares, distance, time, etc), and a set of linear inequalities that constrain the variables in the optimization. A classic example of a linear program is a transport model, such as shipments of coal between producing and consuming regions.
